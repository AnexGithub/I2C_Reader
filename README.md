# I2C_Reader
We can find the I2C address using this code

      Breakout Boards

VCC* - 5V/3V3 pin (depending on breakout module)
GND* - GND
SDA - SDA
SCL - SCL

      PIN CONNECTIONS
 Form factor   	  SDA        SCL  
 
   UNO          SDA/A4      SCL/A5
   Nano           A4         A5
   Pi Pico      GPIO 4      GPIO 5
   ESP 8266     GPIO 4/D2   GPIO 5/D1
   ESP 32       GPIO 21     GPIO 22
